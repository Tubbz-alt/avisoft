<h1>Avisoft</h1>
<p>Construcción de una aplicación para la gestión de manejo y control de una granja avícola, en colaboración con los aprendices del programa de Especialización en Administración de Procesos Productivos Avícolas del SENA.</p>
<ul>
	<li><h2>Modulo Avicola</h2>
		<ul>
			<li>Gestión Granja: Registro de la granja con sus respectivos galpones, sus lotes y además de los equipos requeridos.</li>
			<li>Gestión Engorde: Maneja la tasa de mortalidad y el consumo de alimento de las aves durante las etapas de levante y engorde.</li>
			<li>Gestión Ponedoras: Maneja la tasa de mortalidad, el consumo de alimento y la producción de huevos de las aves durante las etapas de levante y postura.</li>
		</ul>
	</li>
	<li><h2>Modulo Inventario</h2>
		<ul>
			<li>Gestión Proveedores: Realiza inserción, modificación y desactivación de los datos de los proveedores de insumos para la granja.</li>
			<li>Gestión Insumos: Registra, modifica y desactiva los datos de un insumo de la base de datos de la granja.</li>
			<li>Gestión Orden de Compra: Permite generar entradas de los insumos existentes en la granja mediante la creación de una compra.</li>
		</ul>
	</li>
	<li><h2>Modulo Administrativo</h2>
		<ul>
			<li>Reportes de Errores: Envía los datos de las fallas presentadas en la ejecución del programa.</li>
			<li>o	Gestión de Acceso:  Inicia y cierra la sesión del usuario, además recupera la contraseña del usuario.</li>
			<li>o	Configuración de Cuenta: modifica el correo del usuario y la clave.</li>
		</ul>
	</li>
	<li><h2>Reportes</h2>
		<p>A continuación se especificaran algunos reportes generados por la aplicación</p>
		<ul>
			<li>Reporte de Inventarios</li>
			<li>Reporte de Parámetros Semanales (Engorde)</li>
			<li>Reporte de Parámetros Semanales (Ponedor)</li>
		</ul>
	</li>
</ul>
